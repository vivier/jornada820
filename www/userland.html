Linux Userland for Jornada 820

Whereas the kernel is specific to the model, most any Linux distribution for any ARM platform will work on any other ARM platform that has sufficient memory and persistent storage.

You may have a look at userland distributions for the Psion5MX, Zaurus, iPAQ, jornada720, etc. Ask Google, see on handhelds.org, etc. If you have big enough a CF memory card (or a CF harddisk!), you can even install debian/arm!
