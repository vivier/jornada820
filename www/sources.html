Linux for Jornada 820 Source Access

All the code we develop to get Linux running on the Jornada 820 is in CVS; it includes scripts to load and build a cross-compiling toolchain. To consult and tweak the sources of this software, you can use our anonymous CVS access.

NB: as you can see with ViewCVS, the modules to checkout are bootimage and www.
