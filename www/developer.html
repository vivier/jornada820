Developer Information

Linux Bootimage

The Sources on CVS contain scripts that will automatically fetch and compile all the software dependencies used while building the bootimage. The result bootimage plus bootloader executable can easily fit within 2.5MB of vfat partition on a CompactFlash card.

Check our project status page to see what remains to be done, and come discuss on our mailing-lists if you're doing any serious hacking with our code base. You may join the project if you want -- just show us your code.

Bootloader

Our bootloader is hpcboot from NetBSD (compiled binary). Oleg Gusev tailored a modified version for the Jornada 820 -- beg the sources from him!

Toolchain

For now, so as to compile hpcboot, you will need the old version of Embedded Visual C++ 3.0 on ftp.microsoft.com (hint EN_WINCE_EMBDVTOOLS30.exe), it includes the WinCE 2.11 SDK.

GNU binutils support 'arm-wince-pe' since ages, and gcc-3.4.0 is supposed to have the 'arm-wince-pe' target (I have not tested it). The free software solution is not far away, but needs your help ;-)
