Linux for Jornada 820 Project Status

Currently, we can boot and have quite a usable Linux machine using Linux kernel 2.4: the LCD screen, keyboard and mouse, serial device, PC Card and CompactFlash slots are essentially working correctly.

Support is still lacking for the following device: sa1101-fb (external VGA), sa1101-usb, ucb1x00-audio, ucb1x00-dac and ucb1x00-gpio, decoding ser1 (1200 bps) battery datastream, poweroff/sleep, etc.

We have started working on Linux 2.6, but we still don't have working pcmcia, only the LCD, keyboard and mouse (with spurious kernel error messages), and serial (untested). Progress is promising though, so help welcome.
